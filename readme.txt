En la última versión se agregó:
 - El carrito se ve en todas las páginas, y se pueda manipular desde todas las páginas.
 - Si tengo varias pestañas (ej, inicio y merchandising) abiertas al mismo tiempo, y agrego un producto al carrito desde merchandising, en la pestaña inicio se actualiza también. Y si desde la página inicio quito del carrito un producto, se actualiza en la otra pestaña, merchandising el catalago.
 - Mensajes (toast) para cuando se agrega un producto, se quita uno y se finaliza la compra.
 - Los productos se obtienen por fetch desde un archivo json.
 - Se agregaron los contadores de producto al carrito y a cada producto del catalogo.
 - El combo para seleccionar la condición fiscal.
 - El total según la condición fiscal.
 - El formulario de finalizar compra.
